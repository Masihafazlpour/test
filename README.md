# سامانه آزمون حرفه‌ای

نسخه‌ی بازنویسی‌شده برای اجرای تحت وب، با این ویژگی‌ها:

- پاسخ‌برگ ۵ ستونه با چک‌باکس‌های HTML
- پیش‌نمایش ثابت PDF / تصویر در ستون راست
- مدیریت آزمون در مودال آکاردیونی
- ذخیره‌ی تنظیمات و تاریخچه به‌صورت محلی
- ورود کلید از Excel / CSV / ثبت دستی
- گزارش پایان آزمون به‌شکل کارنامه‌ی آماده‌ی چاپ / ذخیره به PDF

## اجرا
فایل `index.html` را در مرورگر باز کن.

## نکته‌ی PDF
دکمه‌ی «دانلود PDF کارنامه» یک گزارش چاپ‌محور باز می‌کند. در پنجره‌ی چاپ، گزینه‌ی **Save as PDF / ذخیره به‌صورت PDF** را انتخاب کن.

## ساختار کلید
فایل نمونه‌ی `question_key_template.xlsx` شامل دو ستون است:

- `question_no`
- `answer`

مقدار پاسخ باید یکی از این‌ها باشد:
`1`, `2`, `3`, `4`

## نمونه‌ی JSON
فایل `sample-exam.json` یک نمونه از ساختار ذخیره‌سازی و خروجی را نشان می‌دهد.
